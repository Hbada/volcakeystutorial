Play real-time and saved patterns

Once patterns are saved to a memory slot, press Memory + M1 through M8 to play them. You could 'perform' your song by recording up to 8 tracks and playing them live, one at a time, with optional real-time sound adjustments described throughout this TinyGuide.

As the sequencer plays your saved pattern, a blinking red dot moves left to right to indicate the step in the sequence that is active

Adjust the sound

Applies to live playing, and adjusting the sound of saved patterns that are playing.

Start Clean

  • Set VCO and LFO sections knobs at zero
  • VCF section all at mid-point
  • Last section: Attack at zero; the others can be higher.

Voice knob

  • Try out Octave, Fifth, etc.
  • Poly mode enables chords

VCF

  • From tinny/airy to rich.
  • Adjust Peak to hear its effect
  • Revisit VCF section if you adjust LFO section

LFO Rate

  • Clockwise: adds repetition to notes held
  • Clean sound: this column all zero level
  • At or near max: fatter sound
  • Adjust Peak to hear its effect
  • Revisit VCF section if you adjust LFO section

LFO Pitch Int

  • Changes dropoff of pitch while key held down
  • Start at zero > increase slowly to gauge results
  • Revisit Pitch Int if you change Rate

LFO Cutoff Int

  • Start at zero
  • Increase to set dropoff of cutoff
  • Revisit if you adjust Rate

Delay Time & Feedback

  • Purpose: delays output > space effect
  • Delay Time: adjust for more or less dreamy sound

Motion recording

  • Allows recording new adjustments to a saved pattern
  • First save a pattern to a memory slot, then play it
  • Hit Record (big dot) then make adjustments while sequencer continually plays

Motion recording surprises

  • While pattern is playing, adjustments may not fully affect recorded notes (usually) even though they're very noticeable when playing live
  • Recording stops automatically after a short time; hit Record again if necessary and wait for the step you need to adjust

One More Note

  • Not all adjustments are eligible for 'motion' recording in this manner. Results will vary, because the Volca tries to quantize or fit the adjustments to the steps of the sequencer; some subtle adjustments may be absent in the recording so you might need to repeat, to record the effect you want.
  • Alternative: record to a computer to adjust a saved pattern in real time.

Real-time recording methods

Record your tune to a memory slot then save for long-term storage.

New recording

1) Choose memory slot, 2) Press Record, 3) Play pattern up to 16 steps. Sequencer continues looping and recording stays active.

Add to stored pattern

1) Choose memory slot, 2) Press Play then Record or both at same time, 3) What you play is recorded even while sequencer loops. New notes overwrite those in memory. Results may be unpredictable.

Alternative: record live playing to a computer or mobile device through an audio interface, then arrange clips with DAW software.

Stop recording

Record and Play buttons toggle on and off, so usually all you need to do is press Record to stop recording, or Play to stop everything.

Save a recording to memory

Pick a memory slot

Keys M1 through M8 correspond to 8 memory slots. Before creating a new pattern to record & save, listen to them all and pick one to record on.

Save what's playing

Hold Function + press Memory. Let go. Press the memory slot you want to save to, using keys M1 through M8. Warning: listen to the stored patterns beforehand so you know which ones are available. The key that's blinking is the current memory slot that's playing now.

Erase current memory slot

Hold Function and press last key marked "All"

More tips
